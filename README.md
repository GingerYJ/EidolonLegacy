# Eidolon 1.12.2 Port

This is a work-in-progress Minecraft 1.12.2 / Cleanroom port of Eidolon.

The current milestone is a clean starter project that can load as an empty mod before gameplay systems are moved over from the 1.19.2 codebase in `../eidolon-1.20x`.

## Status

- Mod id: `eidolon`
- Package: `elucent.eidolon`
- Target: Minecraft 1.12.2 / Cleanroom Loader
- Current phase: empty mod shell
- Original source license: see `LICENSE-EIDOLON.md`

## Local Commands

```powershell
.\gradlew.bat build
.\gradlew.bat runClient
.\gradlew.bat runServer
```
